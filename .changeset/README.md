# Changesets

Hi! This folder has been automatically generated by `@changesets/cli`.

Changesets are used to version and publish packages in this monorepo. When you make a change that should be released, run:

```bash
pnpm changeset
```

Follow the prompts to describe your change. The changeset file is committed alongside your code. On merge to `main`, a GitHub Action opens a "Version Packages" PR; merging that PR publishes the affected packages to npm.
